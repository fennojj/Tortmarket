# Tortmarket
Play-money prediction market for legal outcomes. Forecast verdicts, track campaigns, and compete on the leaderboard.
Tort Market is a play-money prediction game where players use virtual tokens 
to forecast outcomes of civil litigation, legal campaigns, and tort-related events. 
No real money. No wagering. Just skill-based forecasting with leaderboards, 
daily claim bonuses, sponsor integrations, and campaign-driven engagement mechanics.
Built with Expo (React Native) for iOS and Android, backed by Supabase for 
real-time data, authentication, and leaderboard state.
Stack: Expo · React Native · TypeScript · Supabase · Vercel (landing page)
